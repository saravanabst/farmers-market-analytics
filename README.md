# 🌾 Farmers Market Analytics Platform

## Project Overview

This project demonstrates how a production-grade Data Analytics solution is built from scratch using industry-standard tools and practices.

The objective is to build an end-to-end analytics pipeline similar to what a Data Analyst would develop in a real company.

The project covers the complete lifecycle—from data storage and SQL analysis to Python automation, Excel reporting, Tableau dashboards, and business insights.

---

## Business Scenario

A Farmers Market company sells products through multiple vendors across different locations.

The management team wants a centralized analytics platform to answer important business questions such as:

* Which products generate the highest revenue?
* Which vendors perform the best?
* Which categories are growing over time?
* Which cities contribute the most sales?
* What are the monthly sales trends?
* Which products require inventory attention?

The goal is to transform raw operational data into meaningful business insights.
